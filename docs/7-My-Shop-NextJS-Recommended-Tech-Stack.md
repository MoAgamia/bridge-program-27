# Recommended Tech Stack for Beginners

As you continue learning React and Next.js, you'll start using more tools that make development easier.

Don't worry about learning everything at once. Learn each tool when you actually need it.

A very good place to start is looking at Developer Roadmap.
Link: https://github.com/nilbuild/developer-roadmap

---

# 1. Next.js

The framework we'll use to build our applications.

You'll learn:

- Routing
- Layouts
- Server Components
- Client Components
- API Routes
- Image Optimization

---

# 2. TypeScript

TypeScript helps catch mistakes before your application runs.

Benefits:

- Better autocomplete
- Fewer bugs
- Easier to understand code

---

# 3. Tailwind CSS

A utility-first CSS framework that makes styling much faster than writing traditional CSS.

You'll learn:

- Layouts
- Spacing
- Colors
- Responsive design
- Dark mode

---

# 4. shadcn/ui

A collection of ready-made UI components built with Tailwind CSS.

Examples:

- Buttons
- Cards
- Dialogs
- Forms
- Tables
- Accordions
- Dropdowns

Instead of building every component from scratch, you can customize these components to fit your project.

---

# 5. Lucide React

A modern icon library.

Use it for:

- Shopping carts
- Search icons
- User icons
- Settings
- Navigation

---

# 6. Zustand

A simple global state management library.

Perfect for storing:

- Shopping cart
- Logged-in user
- Theme
- Favorites

---

# 7. TanStack Query

Instead of fetching data with `useEffect`, TanStack Query helps you:

- Fetch data
- Cache responses
- Handle loading states
- Retry failed requests
- Refresh data automatically

Once you're comfortable with `fetch()` and `useEffect`, this is the next step.

---

# 8. React Hook Form

Makes working with forms much easier.

Examples:

- Login page
- Register page
- Contact form
- Checkout form

---

# 9. Zod

Validates user input.

Example:

Instead of allowing an empty email field, Zod can display an error message before submitting the form.

It works perfectly with React Hook Form.

---

# 10. Deploy on Vercel

Once your application is finished, you'll want to make it available online.

Vercel is the easiest way to deploy Next.js applications.

### Why Vercel?

- Created by the team behind Next.js
- Free for personal projects
- Automatic deployments from GitHub
- Fast global hosting
- Custom domains
- Preview deployments for every commit

---

# Learning Order

1. Next.js
2. TypeScript
3. Tailwind CSS
4. shadcn/ui
5. Lucide React
6. Zustand
7. TanStack Query
8. React Hook Form
9. Zod

---

# Later (After You're Comfortable)

Once you've built a few projects, you can explore more advanced tools such as:

- Auth.js (Authentication)
- Prisma (Database ORM)
- MongoDB / PostgreSQL
- TanStack Table
- Motion (Animations)
- Recharts / Echarts (Charts)
- Playwright (End-to-End Testing)
- Vitest (Unit Testing)

There's no need to learn these immediately. Focus on mastering the fundamentals first, then gradually add new tools as your projects become more advanced.
